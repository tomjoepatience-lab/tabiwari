### プロモーションテキスト

支出を記録すると、マネコが歩き、街が少しずつ変わります。目標貯金・レポート・家族共有を、ひとつのやさしい家計簿に。

### 説明文

マネコ家計簿は、毎日のお金の記録を「街を育てる体験」に変える家計簿アプリです。

レシートから支出を読み取り、費目を自動で分類。ジャンル別・月次・カレンダー・地図のレポートで、お金の使い方を振り返れます。

目標を決めて貯金すると、マネコが道を進み、街並みも少しずつ変化。ひとりでの利用はもちろん、ふたり・家族の家計簿スペースを作って一緒に記録できます。

主な機能:

- 支出のかんたん記録とレシート読み取り
- 費目の自動分類と手動修正
- 目標貯金とマネコの旅
- ジャンル・月次・カレンダー・地図レポート
- 複数の家計簿スペースと招待URL
- 個人利用・家族利用の切り替え
